# Vika Casino Bot

Персональный Telegram бот с казино-игрой "Рандеву"

## Структура
- `/webapp` - WebApp игра (GitHub Pages)
- `/handlers` - обработчики бота
- `/resources` - статические данные

## Запуск бота
```bash
python3 bot.py
```

## WebApp URL
После деплоя на GitHub Pages: `https://[username].github.io/Vika_slava_bot/webapp/`
